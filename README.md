# 🛒 E-Commerce Sales Dashboard
> An interactive Power BI dashboard built using orders and 
customer data to track sales, profit and customer behavior.

## Objective
To analyze e-commerce sales performance and identify trends 
across states, product categories and payment preferences.

## Dataset
- Orders.csv — transaction level order data
- Details.csv — customer and product details

## Tools Used
- Power BI — dashboard & visualizations
- CSV — raw data

## Key Findings
- Clothing dominates sales with 63% of total quantity sold
- Maharashtra is the highest revenue generating state
- COD is the most preferred payment method at 44%
- Printers generate the highest profit among all sub-categories

## How to View
Download the `.pbix` file and open with Power BI Desktop

## File Structure
- `E_Commerce_Sales_Dashboard.pbix` — Power BI file
- `Orders.csv` — Orders data
- `Details.csv` — Customer data
- `E-Commerce_Sales_DashBoard.pdf` — Dashboard preview
