🛒 E-Commerce Sales Dashboard

An interactive Power BI dashboard built using orders and customer data to track sales performance, profit trends and customer behavior across states and categories.

Built an interactive Power BI dashboard analyzing 438K e-commerce sales across 4 states and 3 product categories.
Visualized monthly and quarterly profit trends across product sub-categories to identify performance patterns.
Evaluated customer payment behavior across 5 payment modes to understand transaction preferences.
